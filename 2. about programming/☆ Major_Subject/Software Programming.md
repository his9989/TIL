* Rest api의 CRUD
  + GET : Resource의 목록 표시
  + GET에 id 지정 : Resource 하나의 내용을 표시 (루트에 들어가는 리소스, id 암호화 필요)
  + POST : Resource 생성
  + PUT: Resource 수정 (루트에 리소스와 id들어감 – 들어가는 리소스, id 암호화 필요))
  + DELETE : Resource 삭제
