### C++ 특징
* overloading : 매개변수의 개수나 타입이 다른 동일한 이름의 함수
<code>

      class Circle{
        Circle();
        Circle(int r);
      }

      Circle::Circle(){
        // 매개 변수 없는 경우
      }

      Circle::Circle(int r){
        // 매개 변수 있는 경우
      }

</code>
