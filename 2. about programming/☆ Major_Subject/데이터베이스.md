#### ACID : DB 트랜잭션이 안전히 수행됨을 보장하는 성질
* 무결성 : 데이터가 정확하고 일관되는 성질
* 원자성 : 트랜잭션과 관련된 작업들이 모두 반영되거나, 전혀 반영되지 않는다.
  + WAL (로그 선행 기법 ) : 원자성의 한 계열
* 일관성 : 데이터는 변하지 않아야된다.
* 독립성 : 트랜잭션 수행 시 다른 트랜잭션의 연산 작업이 끼어들지 못하도록 보장
